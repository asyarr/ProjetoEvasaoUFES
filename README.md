# Projeto Evasão UFES

Este projeto visa identificar os principais fatores associados à evasão dos estudantes do Centro
de Ciências Exatas da Universidade Federal do Espírito Santo, utilizando uma abordagem
quantitativa com ferramentas estatísticas e computacionais avançadas. Dados serão coletados
por meio de questionários baseados na teoria de resposta ao item, garantindo precisão na
avaliação de características individuais e contextuais dos alunos. Informações adicionais,
como desempenho e histórico acadêmico, serão obtidas nas bases institucionais da Ufes.
